proxyの設定メモ
==================================================


プロキシ権限設定
--------------------------------------------------------

権限追加::

   git config --global http.https://github.com/.proxy http://proxy.example.com:8080
   git config --global http.https://bitbucket.org/.proxy http://proxy.example.com:8080
   
ここのhttp://proxy.example.comを自分のプロキシに変更
