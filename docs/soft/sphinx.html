sphinxのインストールメモ¶

sphinxのインストール手順は以下となります。

pythonのインストール¶

ここでは最新版のpython27をインストールした インストール方法は このURL に参考する。

インストールする時、customize installationを選定する
※ インストール中path追加のチェックボックスをチェックすれば、環境設定が不要。

インストール後、バージョン確認：

python -V
  • ライブラリとかのインストール
  1. ez_setup.py をダウンロードして、コマンドラインから実行。
python ez_setup.py
  1. pip（パッケージマネージャ）のインストール
easy_install pip
  1. sphinxのインストール
easy_install sphinx

以下のコマンドで、Sphinxのプロジェクトが作成できればインストールは成功している。

sphinx-quickstart

実行すると、プロジェクトの詳細設定を聞かれるメッセージがでるが、 とりあえずEnterを押していく。途中、以下の３つの項目が入力必須項目。

  • プロジェクト名: sphinx
  • バージョン番号: 1.0
  • 著者の名前: ko

javasphinxのインストール¶

pip install javasphinx

javasphinxをうまくインストールできない場合、エラーを確認して、必要なライブラリを追加する。

たとえば：lxmlの問題なら、lxmlのインストールが必要。

pip install lxml

Tip

lxmlのインストールエラーが出たら、次の方法でインストールする

ここから lxmlのファイルを取得する

例：lxml-3.7.3-cp27-cp27m-win_amd64.whl

以下のコマンドを実行する(win7 64bit)

pip install lxml-3.7.3-cp27-cp27m-win_amd64.whl

うまくインストールできるかをいかのコマンドで確認する

pip freeze

問題がなければ、javasphinxを再インストールする

最後htmlファイルを生成する¶

make html
